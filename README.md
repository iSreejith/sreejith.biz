## üì¶ Installation & Usage

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/sreejith.biz.git
   ```

2. Navigate to the project directory:
   ```bash
   cd sreejith.biz
   ```

3. Open `index.html` in your browser:
   - Double-click the file, or
   - Use a local server (recommended):
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     ```

4. Visit `http://localhost:8000` in your browser

### Deployment

This is a static website and can be deployed to any static hosting service:

- **GitHub Pages**: Push to GitHub and enable GitHub Pages in repository settings
- **Netlify**: Drag and drop the folder or connect your GitHub repository
- **Vercel**: Import your GitHub repository
- **AWS S3**: Upload files to an S3 bucket configured for static website hosting
- **Any web server**: Upload files via FTP/SFTP

## üé® Customization

### Colors

The website uses CSS variables for easy color customization. Edit the `:root` section in the `<style>` tag:

```css
:root {
  --color-black: #111111;
  --color-white: #ffffff;
  --color-grey-light: #f4f4f4;
  --color-grey-medium: #e0e0e0;
  --color-grey-dark: #555555;
  --color-accent: #2563eb;
  --color-accent-hover: #1d4ed8;
}
```

### Content

Update the content directly in the HTML file:

- **Personal Information**: Update name, title, and description in the hero section
- **About Section**: Modify the paragraph in the `#about` section
- **Experience**: Add/edit timeline items in the `#experience` section
- **Expertise**: Update skills in the `#expertise` section
- **Education**: Modify education items in the `#education` section
- **Contact**: Update email and LinkedIn links in the `#contact` section
